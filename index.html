LiveWB

A smart Live Weather Background dynamic changer. LiveWB is a Bash application, which updates Gnome 3 background, using almost real time satelite weather images. LiveWB is autoconfigured and respects both local and remote resources. Custom configuration files may be created easily.

Live weather background

If you want your gnome desktop background to be usefull, checkout this image. By watching regularly LiveWB composite sattelite images, you may observe more details about weather and soon you will be able to make a prediction for the next day. This application is developed in a Debian GNU/Linux system using Free and open-source software (FOSS) and it is licenced under GNU GPLv3.

How does it work

EUMETSAT is providing the meteorological satellite images. LiveWB downloads selected images to the local computer and creates a custom composite image. LiveWB knows how often each type of image is updated. LiveWB also knows when there is a maximized window. So, it downloads and creates a new composite image ONLY when a new image exists AND there are no maximized windows. This is a smart way to respect remote resources, as most of the time a linux hackers are working on a maximized window and thus a background update is not needed. A nice effect is that if there is a maximized window for long enough and you close it (or restore to smaller size) the background image updates within few seconds.
